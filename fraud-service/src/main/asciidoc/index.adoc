[[overview]]
== Overview
The Fraud Detection Service provides a reactive API for real-time risk evaluation of transactions using machine learning models (ONNX/XGBoost).

[[overview-http-verbs]]
== HTTP verbs
Standard HTTP verbs are used.

|===
| Verb | Usage

| `POST` | Used to submit data for evaluation
|===
